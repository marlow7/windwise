# 风电场智慧协同云平台

## 项目概述

基于 Vue3 + Vite + Element Plus + ECharts + Tailwind CSS 构建的风电场智慧协同云平台前端应用，采用浅色清爽工业风主题设计。

## 技术栈

- **框架**: Vue 3.5 + TypeScript
- **构建工具**: Vite 8
- **UI组件库**: Element Plus 2.13
- **图表库**: ECharts 6
- **路由**: Vue Router 5
- **样式**: Tailwind CSS 4

## 项目结构

```
src/
├── App.vue              # 根组件（导航栏 + 路由视图）
├── main.ts              # 入口文件
├── router/index.ts      # 路由配置
├── data/index.ts        # 静态数据配置
├── styles/index.css     # 全局样式
├── vite-env.d.ts        # 类型声明
└── views/
    ├── Dashboard.vue    # 首页大屏
    ├── Overview.vue     # 风机概览
    ├── Monitor.vue      # 风机监测
    ├── Operation.vue    # 运维管理
    └── Data.vue         # 数据管理
```

## 页面说明

### 1. 首页大屏 (/)
- 风机状态饼图（并网、故障、待机等）
- 机型故障预测分布
- 月发电量折线图
- 风机状态时间TOP5柱状图
- 设备健康指数统计
- 风电场地图分布
- 预警风机列表和故障分级
- 累计电量统计表
- 风速功率数据表

### 2. 风机概览 (/overview)
- KPI指标展示
- 故障分布饼图
- 年发电量统计柱状图
- 风机状态统计
- 机组实时数据表格
- 各机组利用小时数折线图
- 风场性能分析柱状图

### 3. 风机监测 (/monitor)
- 机组选择（1-19号）
- 控制系统参数
- 齿轮箱参数
- 变桨系统参数
- 偏航系统参数
- 风力发电机参数
- 24小时风速-功率折线图
- 诊断分析结果
- 剩余寿命预测图表

### 4. 运维管理 (/operation)
- 机组选择（1-19号）
- 运维反馈表格（编号、机组、部件、故障等级、状态等）

### 5. 数据管理 (/data)
- 模型管理菜单（分类、注册、配置）
- 数据表格展示

## 开发命令

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览生产版本
pnpm preview
```

1. 所有图表使用 ECharts 实现，支持 tooltip 交互
2. 顶部时间实时更新，每秒刷新
3. 导航栏路由切换时当前页高亮显示
